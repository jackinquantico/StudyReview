jQuery 개요

* jQuery 란?

기존에 자바스클비트 언어만으로 구현하기에는 복잡했던 구문 작성을 간소화하기 위해 존 레식에 의해 개발됨
JavaScript 기반의 라이브러리 (유용한 기능들의 모음집)
즉, JavaScript 언어로 유용한 함수나 내용들이 이미 정의되어있는 모음집

복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
적은 양의 코드로 빠르고 풍부한 제공 기능

- 장점
1) DOM 요소 (html 태그)와 관련된 스크립트를 보다 쉽게 구현 가능
2) AJAX (비동기식 통신), 이벤트 처리 등 기능들을 폭넓게 지원, 코드도 간결해짐
3) jQuery 와 관련된 확장형 플러그인, 오픈 API, 프레임워크 등을 지원
(차트, 슬라이드, 애니메이션 효과 등)

* jQuery 라이브러리 연결 방법

head 태그 내부에 script 태그로 jQuery 관련 .js 파일로 가져오기 (외부 방식)
오프라인 / 온라인 방식으로 나뉜다.

- jQuery 공식 사이트 : 공식 사이트

1. jQuery 라이브러리 파일을 직접 다운로드 받은 후 경로 지정 (오프라인 방식)
- .js 파일을 직접 다운로드 받아 저장한 후 해당 지정된 파일 경로를 script 태그로 작성
2. CDN (Content Delivery Network) 을 통한 연결 (온라인방식)
- 라이브러리를 제공하는 사이트 url을 제시해서 파일 경로로 지정
- 인터넷이 연결되어있는 상황에서만 사용 가능

* jQuery 구문 작성

jQuery 구문으로 DOM 요소를 다루는 구문 작성할 것이기 때문에 jquery 명령은 문서 상에 요소들이 다 로딩(브라우저 상에 태그들이 랜더링)된 후에 실행되어야 한다.

- 순수 자바스크립트에서의 해당 내용은 

            window.onload = functio() {
                실행내용; // 여기에 작성된 내용은 문서 상의 태그들이 다 로딩된 이후 알아서 실행됨
            }

            onload 함수 : 페이지가 로디되면 자동으로 실행되는 함수
                          단, JavaScript 에서의 window.onload 구문은
                          해당 문서의 요소 뿐만 아니라 연동되어 있는 외부 문서도
                          다 가지고 온 뒤에 실행됨
                          window.onload 는 해당 문서에서 단 한번만 작성 가능하고 단 한번만 실행됨
- jQuery 에서의 해당 내용은 3가지 구문으로 제공 
            1)
            jQuery(document).ready(function() {
                실행내용; // 여기에 작성된 내용은 문서 상의 태그들이 다 로딩된 이후 알아서 실행됨
            }); 

            2)
            $(document).ready(function() {
                실행내용; // 여기에 작성된 내용은 문서 상의 태그들이 다 로딩된 이후 알아서 실행됨
            });

            $ : jQuery 키워드를 나타낸다.

            3)
            $(function() {
                실행내용; // 여기에 작성된 내용은 문서 상의 태그들이 다 로딩된 이후 알아서 실행됨
            });

            jQuery 에서의 해당 function 은 DOM 요소들만 다 로드되면 다 실행됨
            여러 번 기술 가능, 제한 없음.

자바스크립트와 jQuery 방식의 차이점
- 자바스크립트는 속성에 직접 접근해서 대입하는 구문 -> onload 가 단 한번만 실행
- jQuery 는 이미 만들어진 메소드를 호출해서 값을 변경하거나 가져오는 구문 -> 호출할 때마다 실행됨

만약 콘솔에 $ is not defined / jQuery is not defined 가 출력되면 jQuery 라이브러리 연동을 안 한 것
