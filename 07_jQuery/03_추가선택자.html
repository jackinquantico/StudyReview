* 속성선택자

선택자[속성] : 특정 속성을 가지고 있는 모든 요소 선택
선택자[속성=속성값] : 속성값이 일치하는 모든 요소 선택
선택자[속성~=속성값] : 특정값을 단어로 포함하는 모든 요소 선택 (띄어쓰기 기준)
선택자[속성^=속성값] : 속성값이 해당값으로 시작하는 모든 요소 선택
선택자[속성$=속성값] : 속성값이 해당값으로 끝나는 모든 요소 선택
선택자[속성*=속성값] : 속성값에 특정값이 포함되는 모든 요소 선택
