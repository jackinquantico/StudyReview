* 선택된 요소의 속성값을 변경하고자 할 때
- 순수 자바스크립트 방식 : 선택된 요소 객체의 속성에 직접 접근해서 대입하는 방식

- 이벤트 : .on이벤트명 = function() {};

- jQuery 방식 : 해당 속성과 관련된 기능을 제공하는 메소드를 호출하는 방식
- 스타일 : .css("스타일속성명", "변경할 값");
- 내용 : .html("변경할값");
- 이벤트 : .이벤트명(function() {});
